HOW TO RUN:
1. python -m venv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. (Recommended) Redis: docker run -d -p 6379:6379 redis
5. cd backend
6. python main.py
7. Open http://127.0.0.1:5000/

Optional Celery (from backend/):
  celery -A tasks.celery_app.celery_app worker --loglevel=info --pool=solo
  celery -A tasks.celery_app.celery_app beat --loglevel=info

Default admin: admin@ppa.com / admin123
DB is created automatically on first run.

BATCH JOBS / NOTIFICATIONS (local demo):
- Daily reminders: one message per eligible student + admin digest
- Monthly report: HTML report for the report month → admin
- CSV export: async job; student gets alert + browser download
Without SMTP, all notifications are saved as .txt files in:
  backend/notifications/
Optional real email: set SMTP_HOST, SMTP_USER, SMTP_PASSWORD, ADMIN_EMAIL
Optional Google Chat: set GCHAT_WEBHOOK_URL
CSV files are saved in: backend/exports/

IMPORTANT: If an old Flask process is still running, stop it first
(Ctrl+C in that terminal), then start again with: python main.py
